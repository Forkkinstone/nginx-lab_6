## 📌 Описание задания
Цель работы — развернуть многоконтейнерную инфраструктуру в Docker и реализовать взаимодействие PHP-приложения с тремя типами баз данных:
Redis — использование в качестве хранилища для корзины покупок (Key-Value).
ClickHouse — выполнение аналитических запросов к системным таблицам (Column-oriented).
Elasticsearch — индексация и поиск документов (Document-oriented).
Результат доступен по адресу [http://localhost:8080](http://localhost:8080).

---

## ⚙️ Как запустить проект

1. Клонировать репозиторий:
   ```bash
   git clone <ссылка на репозиторий>
   cd nginx-lab
Запустить контейнеры:
```bash
docker-compose up -d --build
```
Открыть в браузере:
```http://localhost:8080```

📂 Содержимое проекта

```docker-compose.yml``` — описание сервиса Nginx

```www/src/``` — исходный код классов для работы с базами данных (RedisExample.php, ClickhouseExample.php, ElasticExample.php).

```www/index.php``` — главная страница с демонстрацией работы всех сервисов.

```www/ClientFactory.php``` — фабрика для создания HTTP-клиентов Guzzle.
